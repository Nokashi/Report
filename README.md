# ST-Visions Documents

This repository contains the **LaTeX source files** for the Bachelor's Thesis:

ST-Visions+: Extending the ST-Visions Library for Real Time Visual Analytics, Paraschos Moraitis

The project includes all chapters, bibliography files, figures, custom packages, and build configuration needed to compile the final thesis document.

## Template

This LaTeX project is based on the **NTUA ECE Thesis Template** created and maintained by *estamos*.  
You can find the original template here:
https://github.com/estamos/NTUA-ECE-Thesis-Template

The structure, class files, and formatting rules are derived from this template, with additional customizations specific to my thesis.
